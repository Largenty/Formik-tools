# Formik Tools

Simple re-usable components using :

- Formik
- Yup
- DatePicker

# Input type view in the project :

- Input
- Checkbox
- Radio
- Dropdown
- DatePicker
- Textarea

# No CSS:

There is no css.

# NVM version :

For this project you need to downgrade your node version to 14 min.
use :
`nvm use 14`

Enjoy it ! :)
